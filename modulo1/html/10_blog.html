¿Qué es HTML5?

HTML5 es la quinta versión del estándar de marcado para la web. Fue publicada oficialmente por el W3C en 2014 y desde entonces es el estándar para crear páginas web.

HTML es la base de toda la web. Es el lenguaje con el que describimos la estructura y el significado del contenido digital.

Principales novedades de HTML5

  • Nuevas etiquetas semánticas: <header>, <nav>, <main>, <footer>
  • Soporte nativo de vídeo y audio sin plugins
  • El elemento <canvas> para gráficos 2D
  • Nuevos tipos de input en formularios: email, date, range...
  • Almacenamiento local con localStorage y sessionStorage

Pasos para aprender HTML5

  1. Entender la estructura básica de un documento
  2. Aprender las etiquetas de texto y formato
  3. Practicar con enlaces e imágenes
  4. Crear tablas y formularios
  5. Usar etiquetas semánticas correctamente
  6. Integrar CSS para el diseño visual

Comparación de versiones

HTML 4 vs HTML5

Con HTML 4 era común usar tablas para maquetar el layout. Con HTML5 usamos flexbox y grid en CSS para eso.

Fórmula del éxito
