## Glosario de Etiquetas HTML Usadas en el Proyecto

A continuación, un diccionario ordenado alfabéticamente de las etiquetas HTML utilizadas en los archivos del proyecto del menú de restaurante. Cada etiqueta incluye una descripción directa de su función y un ejemplo relacionado con el negocio.

- **<abbr>**: Define una abreviatura con expansión en `title`. Ejemplo: `<abbr title="Restaurante Gourmet">RG</abbr>` para abreviar el nombre del restaurante.
- **<b>**: Pone el texto en negrita. Ejemplo: `<b>Precio especial</b>` para resaltar ofertas en el menú.
- **<blockquote>**: Crea una cita en bloque. Ejemplo: `<blockquote cite="reseña">Excelente comida</blockquote>` para reseñas de clientes.
- **<body>**: Contiene el contenido visible de la página. Ejemplo: Todo el menú del restaurante va aquí.
- **<cite>**: Indica el título de una obra. Ejemplo: `<cite>Libro de Recetas Gourmet</cite>` para citar libros de cocina.
- **<dd>**: Define la descripción en una lista de definiciones. Ejemplo: `<dd>Plato principal con carne</dd>` en el glosario del menú.
- **<del>**: Indica texto tachado (eliminado). Ejemplo: `<del>$30</del>` para precios antiguos en promociones.
- **<dl>**: Crea una lista de definiciones. Ejemplo: Glosario de términos del restaurante.
- **<dt>**: Define el término en una lista de definiciones. Ejemplo: `<dt>Entradas</dt>` para categorías del menú.
- **<em>**: Pone el texto en cursiva para énfasis. Ejemplo: `<em>Recomendado</em>` para platos destacados.
- **<footer>**: Define el pie de página. Ejemplo: `<footer>© 2026 Restaurante Gourmet</footer>` para copyright.
- **<h1> a <h6>**: Crea títulos; el número indica el tamaño (1 más grande, 6 más pequeño). Ejemplo: `<h1>Menú del Restaurante</h1>` para el título principal.
- **<head>**: Contiene metadatos invisibles. Ejemplo: `<title>Menú Gourmet</title>` para el título de la pestaña.
- **<header>**: Define el encabezado. Ejemplo: `<header>Restaurante Gourmet</header>` para el logo y nombre.
- **<hr>**: Inserta una línea horizontal. Ejemplo: Separar secciones como entradas y platos principales.
- **<html>**: Elemento raíz de la página. Ejemplo: Envuelve todo el contenido del sitio del restaurante.
- **<ins>**: Indica texto insertado (subrayado). Ejemplo: `<ins>$25</ins>` para precios nuevos en ofertas.
- **<li>**: Define un elemento de lista. Ejemplo: `<li>Pasta Carbonara</li>` en la lista de platos.
- **<main>**: Contiene el contenido principal. Ejemplo: La lista completa del menú.
- **<mark>**: Resalta texto con fondo amarillo. Ejemplo: `<mark>Plato del día</mark>` para destacar ofertas.
- **<meta>**: Proporciona metadatos. Ejemplo: `<meta charset="UTF-8">` para caracteres especiales en nombres de platos.
- **<ol>**: Crea una lista numerada. Ejemplo: Pasos para preparar una receta en el menú.
- **<p>**: Define un párrafo. Ejemplo: `<p>Descripción del plato estrella.</p>` en el menú.
- **<q>**: Crea una cita corta. Ejemplo: `<q>Delicioso</q>` para citas de clientes.
- **<small>**: Reduce el tamaño del texto. Ejemplo: `<small>Precios sujetos a cambio</small>` en el pie.
- **<strong>**: Pone el texto en negrita fuerte. Ejemplo: `<strong>Ingredientes frescos</strong>` para resaltar calidad.
- **<sub>**: Crea subíndice. Ejemplo: Calorías: 500<sub>kcal</sub> en descripciones de platos.
- **<sup>**: Crea superíndice. Ejemplo: Precio: $25<sup>USD</sup> en el menú.
- **<title>**: Define el título de la página. Ejemplo: `<title>Menú Gourmet</title>` para la pestaña del navegador.
- **<ul>**: Crea una lista con viñetas. Ejemplo: Lista de ingredientes en recetas.
- **<!DOCTYPE html>**: Declara que es HTML5. Ejemplo: Al inicio de cada página del restaurante.

## Próximos Elementos HTML a Aprender

Como sigues aprendiendo HTML, aquí un diccionario de elementos avanzados que puedes explorar a continuación, con ejemplos relacionados con el menú de restaurante:

- **<a>**: Crea un enlace. Ejemplo: `<a href="reservas.html">Reservar mesa</a>` para enlazar a página de reservas.
- **<form>**: Define un formulario. Ejemplo: `<form><input type="text" placeholder="Nombre"><button>Enviar pedido</button></form>` para pedidos en línea.
- **<img>**: Inserta una imagen. Ejemplo: `<img src="plato.jpg" alt="Pasta Carbonara">` para fotos de platos en el menú.
- **<table>**: Crea una tabla. Ejemplo: `<table><tr><th>Plato</th><th>Precio</th></tr><tr><td>Pasta</td><td>$15</td></tr></table>` para lista de precios.
- **<video>**: Inserta un video. Ejemplo: `<video src="cocina.mp4"></video>` para videos de preparación de platos.
- **<audio>**: Inserta audio. Ejemplo: `<audio src="musica.mp3"></audio>` para música de fondo en el sitio.
- **<article>**: Define un artículo independiente. Ejemplo: `<article><h2>Reseña del Mes</h2><p>Excelente experiencia</p></article>` para reseñas.
- **<section>**: Define una sección. Ejemplo: `<section><h2>Entradas</h2><ul>...</ul></section>` para organizar el menú.
- **<nav>**: Define navegación. Ejemplo: `<nav><a href="#entradas">Entradas</a></nav>` para menú de navegación interna.
- **<aside>**: Contenido lateral. Ejemplo: `<aside>Horarios de apertura</aside>` para información adicional.
